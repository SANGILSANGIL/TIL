모두 시멘틱을 위한 태그들.
em / strong에 적용되는 표현도 css로 표현 가능하지만 
시멘틱을 위해서 태그들을 분리하였음


emphasized: 문장 / 문자의 강조, 이탤릭

strong: 문장 / 문자의 강조, 볼드  (검색엔진에서 중요시 여김)

definition term: 용어정의

computer code: 컴퓨터코드

sample output from computer program: 예제

keyboard input: 키보드 입력 (ctrl+z) 와 같이 그냥 문자 

variable: 변수 (n의 제곱은?)
